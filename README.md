# ContactsBook
##### A contacts book app with a built in search.

#### ![](contactsbook.gif)

## Built With
React 16.13.0
https://randomuser.me/ API

### How To Use:
  Step 1:
```sh
git clone https://github.com/lina789/ContactsBook.git
```
  Step 2:
```sh
npm install
```
  Step 3:
```sh
npm start
```
